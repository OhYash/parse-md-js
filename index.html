Introduction & Usage

parsemsg.js is a really simple tool that converts app formatted message to HTML. It's built using regular expressions in combination with functional Javascript. It's really tiny (about 150 lines of code) and supports common popular platforms (Whatsapp, Youtube, Slack, and Tenor.cards). Plus, it can be easily extended by adding more regular expressions. Most common apps message format syntax features are supported. To check the recognized syntax rules, have a look at the README.

To use the parser, just grab the javascript code from here and simply call one of the following functions:

var parsedWA = parseWhatsappMessage(yourText);
var parsedYT = parseYoutubeMessage(yourText);
var parsedSlack = parseSlackMessage(yourText);
var parsedTC = parseTenorCardsMessage(yourText);

for the parser to parse the contents of yourText as HTML.
